Enciclopédia Rick and Morty

Uma aplicação web desenvolvida com **HTML**, **CSS** e **JavaScript** que consome a API pública do **Rick and Morty** para pesquisar informações sobre personagens, episódios e localizações da série.

Funcionalidades

* 🔍 Buscar personagens pelo ID
* 📺 Buscar episódios pelo ID
* 🌍 Buscar localizações pelo ID
* 🖼️ Exibir imagem e informações do personagem
* ⚠️ Tratamento de erros para IDs inválidos
* 📱 Interface simples e responsiva

 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* API Rick and Morty

 ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

2. Entre na pasta do projeto:

```bash
cd SEU-REPOSITORIO
```

3. Abra o arquivo **index.html** em qualquer navegador.

Não é necessário instalar dependências ou utilizar um servidor.

## 🌐 API utilizada

Este projeto utiliza a API pública Rick and Morty:

https://rickandmortyapi.com/
 Como utilizar

Digite o ID desejado em um dos campos abaixo e clique em **Buscar**:

 Personagem
 Episódio
 Localização

Exemplos de IDs válidos:

| Tipo        | Exemplo |
| ----------- | ------- |
| Personagem  | 1       |
| Episódio    | 1       |
| Localização | 1       |
